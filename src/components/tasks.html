Testing Tasks

Explore the three key tasks that participants performed during the usability testing sessions.

Task 1: Join a Volunteer Campaign

  1. Log in as User
  2. Navigate to the "Contribute" page
  3. Join a volunteer opportunity

Task 2: Approve or Reject Requests

  1. Log in as Charity
  2. Go to the dashboard
  3. Select a pending name and approve/reject

Task 3: Create a New Campaign

  1. Log in as Charity
  2. Go to "Contribute"
  3. Click "Add Request"
  4. Fill out the form and post the request
